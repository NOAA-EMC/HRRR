Convention Observation Data Assimilation Monitoring using CMon 2.0

This set of conventional obs data assimilation has been created using version 2.0 of the conventional monitoring package (CMon). The principle change from 1.0 is an entire reconfiguration of the CMon package into a vertical structure and separating the data extraction and image generation components of the package.
