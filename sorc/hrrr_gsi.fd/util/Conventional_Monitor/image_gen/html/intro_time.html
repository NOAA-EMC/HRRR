The purpose of monitoring the data assimilation system is to ensure the proper performance of the assimilation system and to diagnose problems with the system for future improvements.

The time series plots include:
  • number of observations
  • number of observations rejected by gross check
  • number of observations rejected by variational quality control
  • number of observations monitored

The plots are made for 10 regions:
  • global(90S-90N, 0-360E)
  • Northern Hemisphere (20N-80N, 0-360E)
  • Southern Hemisphere (20S-80S, 0-360E)
  • Tropics (20S-20N, 0-360E)
  • USA (25N-50N, 125W-65W)
  • Canada (50N-90N, 125W-65W)
  • North & Central America (0N-90N, 165W-60W)
  • South & Central America (0S-90S, 165W-60W)
  • Europe (35N-70N, 10W-25E)
  • Asia (5N-45N, 65E-145E)

The 13 vertical levels are:
  • 0-2000mb
  • 1000-2000mb
  • 900-1000mb (obs type q 950-1000mb)
  • 800-900mb (obs type q 900-950mb)
  • 600-800mb(obs type q 850-900mb)
  • 400-600mb(obs type q 800-850mb)
  • 300-400mb(obs type q 750-800mb)
  • 250-300mb(obs type q 700-750mb)
  • 200-250mb(obs type q 600-700mb)
  • 150-200mb(obs type q 500-600mb)
  • 100-150mb(obs type q 400-500mb)
  • 50-100mb(obs type q 300-400mb)
  • 0-50mb(obs type q 0-300mb)
